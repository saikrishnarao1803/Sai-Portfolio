# Sai Krishna Rao — Portfolio (Static Site)

This is a simple static portfolio website (HTML/CSS/JS). It is **GitHub Pages-ready**.

## Quick start (local)
1. Download / unzip
2. Open `index.html` in a browser

## Publish on GitHub Pages
1. Create a GitHub repo: `sai-portfolio` (or any name)
2. Upload these files to the repo root:
   - `index.html`
   - `styles.css`
   - `script.js`
3. GitHub → **Settings** → **Pages**
4. Source: **Deploy from a branch**
5. Branch: `main` → `/ (root)` → Save
6. Your site URL will appear on the same page.

## Customize (recommended)
- Add your LinkedIn + GitHub links in the Contact section
- Replace “coming” links in Projects with real repo/demo links
- Add a `resume.pdf` file in repo root and add a “Download Resume” button
